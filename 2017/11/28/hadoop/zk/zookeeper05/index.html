zookeeper客户端相关操作

2017-11-28

启动zk客户端：

zkCli.sh
查看帮助命令：help

创建节点：（必须要添加数据才能创建成功）

创建节点：create /huang 26（huang：节点名，26：数据）
创建临时节点： create -e /huang 26（huang：节点名，26：数据）
创建带序号的节点： create -s /huang 26（huang：节点名，26：数据）

查看节点目录结构：

ls / 或者 ls2 /（详细信息）
查看节点信息（不包含数据）：stat /huang

监听节点变化：（以下两种方式都是一次监听）

监听子节点变化：ls / watch
监听当前节点变化：get /huang watch

修改节点内容：

set /huang 27

删除节点：

delete /huang
递归删除：rmr /huang
