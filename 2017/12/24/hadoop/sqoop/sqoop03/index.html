sqoop脚本执行

2017-12-24

创建脚本job_hdfs2mysql.opt

编写脚本：

export
–connect
jdbc:mysql://192.168.127.81:3306/sqoop_db
–username
root
–password
root
–table
user
–num-mappers
1
–export-dir
/sqoop/hive/user
–input-fields-terminated-by
“\t”

注意：要先创建mysql表（user）；

执行脚本：

bin/sqoop \
–options-file /root/sqoop_job/job_hdfs2mysql.opt
